# E-commerce Sales Analysis Dashboard (Power BI)

##  Project Overview
This project involves a comprehensive analysis of the **Olist Brazilian E-commerce dataset**. The goal is to provide actionable business insights into sales performance, customer behavior, and product profitability using Power BI.

##  Key Business Questions Addressed
* What is the total revenue and order volume?
* Which product categories generate the most sales?
* How does sales revenue fluctuate month-over-month?
* Which regions (states) have the highest customer concentration?

##  Tech Stack & Tools
* **Tool:** Power BI Desktop
* **Data Source:** Kaggle (Olist E-commerce Dataset)
* **Data Modeling:** Star Schema
* **DAX:** Used for complex measures and calculated columns.

##  Key Features of the Dashboard
* **Interactive Slicers:** Filter data by Year, Quarter, and Product Category.
* **Sales Trends:** Monthly revenue visualization to identify seasonal peaks.
* **Geographical Mapping:** Analysis of sales distribution across different Brazilian states.
* **Performance Metrics:** Real-time tracking of Total Revenue ($13.59M) and Total Orders (96.48K).

##  Data Modeling
The project uses a **Star Schema** to connect multiple relational tables including:
* Customers Table
* Orders & Order Items Table
* Payments & Reviews Table
* Products Table

##  Key Insights
1. **Revenue Drivers:** Top categories like Health & Beauty and Housewares contribute significantly to total earnings.
2. **Growth Trends:** Identified specific months where marketing campaigns led to a spike in order volume.
3. **Customer Satisfaction:** Analyzed the average review score (3.67) to identify areas for service improvement.
